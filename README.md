# dotfiles
Personal config files


### Install
1. Clone repo
2. Install all the packages
`./install.sh`
3. Symlink all the configs
`./symlink.sh`

#### Polybar install
```
cd ~/polybar-3.5.5 && ./build.sh
```

Answer `No` on the first one (we want to use cmake) and all others `Yes`.

And `Yes` on `sudo make install`
